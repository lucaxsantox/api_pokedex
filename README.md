## Sobre o projeto
É uma aplicação em NodeJS que se alimenta da <a href="https://pokeapi.co/"> </a> e retorna os Pokemons de acordo com o nome ou número deles.

	
## Tecnologias
Atualmente rodando com:

- [x] NodeJS
- [x] Express
- [x] CommonJS
- [x] Nodemon
- [x] Body-Parser
